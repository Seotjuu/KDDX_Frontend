# KDDX_Frontend

## KDDX_Frontend : React.js ( Visual Studio Code )

npx create-react-app KDDX_Frontend ( 프로젝트 이름 )

npm i ( 라이브러리 : d3, bootstrap, fortawesome, axios )

npm start ( Chrome Web Browser 실행 )

### Axios backend data handling to Django

Axios :
    get  : 데이터 조회
    post : 데이터 전송
    put  : 데이터 수정
